# Tetris Redbull

Clon del clásico Tetris construido con Python. Incluye un bot de visión por computadora que juega automáticamente usando análisis de pantalla.

## Stack

Python · Pygame · OpenCV (visión)

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `tetris_phone_bot.py` | Juego de Tetris principal |
| `vision.py` | Bot de visión que analiza el estado del tablero y juega automáticamente |

## Setup

```bash
pip install pygame opencv-python
python tetris_phone_bot.py    # Jugar manualmente
python vision.py              # Activar el bot automático
```
